CSS Image - Wrapping and constraining

Problems

  • Need to make images responsive(fluid)
  • Need to keep the aspect ratio of images
  • Images have its own modeling style
    • (it keeps its aspect ratio and try to expand not fit)

Solution

  • Set a wrapper for an image to manipulate images
  • Give an image inside a wrapper 'width: 100%' to make it fluid
  • Use 'aspect-ratio' to keep the aspect ratio
  • Edit an image for the best intended result
  • Use 'overflow' and 'object-*' properties to edid visual style of image

CSS Image - Use image as darken background

  • Set wrapper to have background color with black
  • Then make the image inside wrapper transparent with opacity
